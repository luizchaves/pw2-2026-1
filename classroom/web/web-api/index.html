Exemplos de Web APIs

  • DOM API Modais e alertas nativos com o elemento <dialog>.
  • Canvas API Desenho de formas, textos e linhas no elemento canvas.
  • Clipboard API Copiar e colar texto usando a área de transferência.
  • Drag and Drop API Arrastar e soltar itens entre colunas de um quadro Kanban.
  • Fullscreen API Entrar e sair do modo tela cheia na página ou em um elemento.
  • Geolocation API Obter a localização atual do usuário com mapa interativo.
  • Fetch API Requisições HTTP (GET e POST) para uma API REST.
  • Web Storage API Armazenar e gerenciar dados no localStorage.
  • Notification API Enviar notificações do navegador com permissão do usuário.
  • Web Speech API Síntese de fala com controle de voz, velocidade e tom.
